Steps to create a repo locally and push to github

1 Local folder
	1.1 local folder created
	1.2 $ git init i# in the folder
	1.3 README.md copied from skills-introduction-to-github
	1.4 $ git add README.md # staged
	1.5 $ git commit # stages committed
	1.6 $ git status # all clean
2 Authenticate
	2.1 $ gh auth login
	2.2 follow on screen guide
3 Create and push repo from local
	3.1 $ gh repo create 
	3.2 follow on screen guide at the end script asks if push need
